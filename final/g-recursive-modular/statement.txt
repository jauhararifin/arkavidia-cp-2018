G. Random Generator

Time Limit : 2 Detik
Memory Limit : 256 MB

[Deskripsi]

Bocan iseng membuat formula untuk random generator buatannya, yakni menggunakan rekurens:

a_n = p * a_{n-1} + q

Kini, ia memiliki bilangan x. Bocan penasaran, berapa n terkecil supaya a_n mod m = x?

[Format Input]
Baris pertama berisi bilangan bulat p, q, dan a_0 dipisahkan dengan spasi.
Baris kedua berisi bilangan bulat m dan x.

[Format Output]
Keluarkan satu baris berisi bilangan bulat n terkecil yang mungkin. Jika tidak ada, keluarkan -1.

[Contoh Input 1]
2 1 5
5 7

[Contoh Output 1]
3

[Contoh Input 2]
2 0 6
1 2

[Contoh Output 2]
-1

[Contoh Input 3]
7 9 16
11 5

[Contoh Output 3]
0

[Batasan]
- 2 <= m <= 10^12
- 0 <= p, q, a_0, a_n <= m-1
- m dijamin merupakan bilangan prima

[Penjelasan]
Pada contoh pertama, barisan yang terbentuk adalah 5, 11, 23, 47, 99, dst. Pada n = 3, 47 mod 7 = 5. Tidak ada nilai n lebih kecil yang memenuhi.

Pada contoh kedua, tidak ada n yang memenuhi.

Pada contoh ketiga, nilai a_0 sudah memenuhi.
